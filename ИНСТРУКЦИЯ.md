# Инструкция по запуску бота "Режим дня и питание"

## Шаг 1. Получи токен бота

1. Открой Telegram, найди бота **@BotFather**
2. Напиши команду `/newbot`
3. Придумай имя бота (например: "Мой трекер дня")
4. Придумай username бота — должен заканчиваться на "bot" (например: `elena_daily_tracker_bot`)
5. BotFather пришлёт тебе **токен** — длинная строка вида:
   `7123456789:AAH1a2B3c4D5e6F7g8H9i0J1k2L3m4N5o6P`
6. Скопируй этот токен

## Шаг 2. Вставь токен в код

Открой файл `bot.py`, найди строку:
```python
BOT_TOKEN = "ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН"
```
Замени на свой токен:
```python
BOT_TOKEN = "7123456789:AAH1a2B3c4D5e6F7g8H9i0J1k2L3m4N5o6P"
```

## Шаг 3. Запусти бота (выбери один вариант)

### Вариант А — локально на компьютере (для теста)

1. Установи Python (если ещё нет): https://www.python.org/downloads/
2. Открой терминал/командную строку в папке с ботом
3. Установи библиотеку:
   ```
   pip install -r requirements.txt
   ```
4. Запусти бота:
   ```
   python bot.py
   ```
5. Бот работает, пока запущен терминал. Открой Telegram, найди своего бота по username, нажми "Старт"

### Вариант Б — бесплатный хостинг Railway (работает 24/7)

1. Зарегистрируйся на https://railway.app (можно через GitHub)
2. Создай новый проект → "Deploy from GitHub repo" (или загрузи файлы напрямую)
3. Загрузи туда файлы `bot.py` и `requirements.txt`
4. В настройках проекта добавь переменную окружения (Environment Variable):
   - Имя: `BOT_TOKEN`
   - Значение: твой токен
5. В коде замени строку токена на:
   ```python
   import os
   BOT_TOKEN = os.environ.get("BOT_TOKEN")
   ```
6. Railway автоматически запустит бота, и он будет работать постоянно

## Что умеет бот

- 🍽 Записывать приёмы пищи
- 💧 Записывать количество воды
- 😊 Оценивать настроение (1-5)
- ⚡ Оценивать энергию (1-5)
- 🏃 Записывать активность
- 😴 Записывать время сна (отбой/подъём)
- 📊 Показывать сводку за сегодня

Все данные сохраняются в файл `daily_tracker.db` (база данных SQLite) — она создаётся автоматически при первом запуске.
